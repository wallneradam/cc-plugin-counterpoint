Disable auto-consult by calling the `mcp__counterpoint__auto_consult_off` MCP tool.

**Auto-consult mode is now OFF.** You no longer need to automatically consult Codex on every decision. Return to normal behavior — only use counterpoint when explicitly invoked via `/counterpoint` or `/consult`, or when you judge a decision truly warrants a second opinion.
