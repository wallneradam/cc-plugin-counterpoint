allowed-tools: mcp__counterpoint__consult_off

Disable auto-consult by calling the `mcp__counterpoint__consult_off` MCP tool.

**Auto-consult mode is now OFF.** Return to manual mode — counterpoint is only invoked when the user explicitly asks (`/counterpoint`, `/consult`, or a direct request for a second opinion). Do not consult Codex on your own judgement.
